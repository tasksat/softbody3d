# softbody3d

A browser-based 3D soft body simulation demo implemented with TypeScript, Three.js, and Vite.

This project is inspired by Matthias Müller’s **[Ten Minute Physics](https://matthias-research.github.io/pages/tenMinutePhysics/index.html)** demo "12 100x speedup for soft body simulations".

**Reference implementation:**

https://github.com/matthias-research/pages/blob/master/tenMinutePhysics/12-softBodySkinning.html

The goal of this project is to rebuild the core idea in a more modular TypeScript codebase.
